Ключи конфигурации:
1. path - путь к папке, в которой должны лежать скрипты
2. encoding - кодировка скриптов

Правила использования:
1. Чтобы утилита отработала правильно, нужно расположить файли со скриптами в указанной в конфигурации папке (ключ "path") в том порядке, в котором они должны выполниться.
2. Для выполнения скриптов для заполнения таблиц Cities, Regions, Countries использовать кодировку Windows-1250 (Central Europe).
3. Указать строку подключения к своей базе данных.
